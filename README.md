# Insta clone App Backend


## Features

- User authentication (signup/login)
- User profiles
- Follow/Unfollow users
- Create, like, and comment on posts
- Fetch user feed

---

## Tech Stack

- Node.js
- Express.js
- MongoDB & Mongoose
- dotenv
- CORS

---

## Installation

### Prerequisites

- MongoDB (local or cloud)
- npm
